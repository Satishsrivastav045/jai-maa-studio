# Jai Maa Bhadrakali Studio Website 🎨

A fully responsive studio services website built with Django 
and deployed live on Render.

## 🔗 Live Demo
[jai-maa-studio.onrender.com](https://jai-maa-studio.onrender.com)

## 🛠️ Tech Stack
- Backend: Python, Django
- Frontend: HTML, CSS, JavaScript
- Database: SQLite / PostgreSQL
- Deployment: Render

## ✨ Features
- Responsive UI for studio services
- Gallery and contact sections
- Clean and professional design
- Live deployment

## 🚀 How to Run Locally
```bash
pip install -r requirements.txt
python manage.py runserver
